# TDD Лабораторная работа - Вариант 16

## Описание
Реализация функции `calculate_discount` с использованием методологии Test-Driven Development (TDD).

## Функциональность
Функция `calculate_discount(price, discount_percent)`:
- Принимает исходную цену и процент скидки
- Возвращает итоговую цену после применения скидки
- Проверяет, что скидка находится в диапазоне от 0 до 100%

## Требования
- Python 3.8+
- pytest
